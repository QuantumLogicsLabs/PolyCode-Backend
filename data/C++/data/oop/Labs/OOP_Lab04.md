## Objectives

- Default Constructor

- Display Function
- Destructor

---

## Task – Mystery Case (Scooby-Doo)

The mystery-solving team led by Scooby-Doo has solved multiple mystery cases.

Design a C++ program that defines a class named `MysteryCase` with the following specifications:

---

### Class Requirements

**Data Members:**

- `caseID` *(int)* — Unique identification number of the mystery case
- `rewardAmount` *(float)* — Reward earned after solving the case

---

**Member Functions:**

Your class must include:

1. **Default Constructor**
   - Initialize data members with default values.

2. **Parameterized Constructor**
   - Initialize data members using arguments passed to the constructor.

3. **Copy Constructor**
   - Create a new object by copying values from another existing object.

4. **Display Function**
   - Display the case ID and reward amount.

5. **Destructor**
   - Display a message when the object is destroyed.

---

### Main Function Requirements

- Create one object using the **default constructor**.
- Create one object using the **parameterized constructor**.
- Create one object using the **copy constructor**.
- Call the display function for each object.
