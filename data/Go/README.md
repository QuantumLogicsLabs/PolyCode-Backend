# Go Learning Guide

This project is a hands-on guide for learning Go with short notes and runnable code.

## Structure

- `lessons/` contains Markdown lessons.
- `examples/` contains Go programs you can run and edit.

## Learning Path

1. Start with [Lesson 1](./lessons/01-introduction.md)
2. Continue through the lessons in order
3. Run the matching example for each lesson
4. Change the code and observe what happens

## How To Run

Run any example from the project root:

```bash
go run ./examples/01-hello
go run ./examples/02-variables
go run ./examples/03-control-flow
go run ./examples/04-functions
go run ./examples/05-collections
go run ./examples/06-structs-interfaces
```

## Suggested Routine

- Read one lesson
- Run the matching example
- Modify the example
- Write your own version from memory
